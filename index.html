Usage

Use images by prepending a @ before the filename. Lines starting with # will be ignored. A presentation file could look like this:

wsent
takahashi
why?
@http://example.com/image.png
easy to use
no dependencies
~100 lines of code
how?
browse to FILENAME
one slide per line
# This is a comment and will not be part of the presentation
# The next line starts with a whitespace, it will not produce an image slide
 @http://example.com/image.png
thanks / questions?

Move between slides with [back]space.
